## Run Locally

Clone the project

```bash
  git clone https://github.com/agus-triono08/LandingPage-KGTPlans.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run watch
```
Open http://localhost:5500 to view it in the browser.
